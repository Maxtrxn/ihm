Pour compiler le programme:
-Sous Windows:
	-Ouvrir un cmd dans le dossier "code"
	-Lancer la commande suivante 'javac -d bin -cp ".;..\libs\json-20231013.jar;..\resources" .\src\Main.java'
	en remplaçant javac par votre commande pour compiler un programme javafx
-Sous Linux:
	-Ouvrir un cmd dans le dossier "code"
	-Lancer la commande suivante 'javac -d bin -cp ".:../libs/json-20231013.jar:../resources" ./src/Main.java'
	en remplaçant javac par votre commande pour compiler un programme javafx
